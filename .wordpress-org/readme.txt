== Description ==

This plugin enables you to process payment cards on your WooCommerce store

### Features

* Easy configuration
* Automatic captcha
* Automatically sets transaction type
* Full transaction log
* Lightweight - Plugin uses optimized autoloader, and minified assets

*WooCommerce logo used with permission*

== Installation ==

1. Upload wc-serbian-nestpay.zip to plugins via WordPress admin panel, or upload unzipped folder to your plugins folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to WooCommerce->Settings to manage the options
